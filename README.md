## GA4GH Data Use and Researcher ID work stream

The DURI ([GA4GH::DURI](http://ga4gh-duri.github.io)) Work Stream is one of the elements of the Global Alliance for Genomics and Health ([GA4GH](http://ga4gh.org)), aimed at developing standard formats for the exchange of genomic data for research and biomedical applications. The GA4GH DURI workstream is led by [Moran Cabili](https://www.linkedin.com/in/moran-cabili-kalmar-4727303/) and [Tommi Nyronen](https://www.linkedin.com/in/nyronen/) All members of this team must follow the [GA4GH Standards of Professional Conduct]( https://www.ga4gh.org/docs/GA4GH-Standards-for-Professional-Conduct_22-Jan-2018.pdf).
